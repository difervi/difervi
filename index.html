Difervi Fast Food

"Fast Food" es un proyecto destacado en mi portafolio que refleja mi habilidad para crear aplicaciones web modernas y funcionales. Este proyecto fue desarrollado utilizando una combinación de tecnologías clave, incluyendo HTML5, Sass, Bootstrap y JavaScript. La característica principal de este proyecto es la implementación de un carrito de compras, diseñado y construido de forma personalizada, utilizando la plataforma Shopify.
